## 📂 FE 폴더 구조

```
📂 src  
├─ 📂 api             // 백엔드 서버와 통신을 위한 API 요청 모듈  
├─ 📂 app             // 페이지와 API 경로 관리  
│   ├─ 📂 api         // API 경로 핸들러  
│   ├─ 📂 auth        // 유저 관련 페이지  
│   ├─ 📂 chat        // 채팅 관련 페이지  
│   └─ ...  
├─ 📂 components      // UI 컴포넌트  
│   ├─ 📂 chat        
│   ├─ 📂 form        
│   └─ ...  
├─ 📂 constants       // 상수 관리  
├─ 📂 containers      // 페이지 컨테이너 컴포넌트  
├─ 📂 font 
├─ 📂 hooks           // 커스텀 훅  
├─ middleware.ts      // 미들웨어 설정  
├─ 📂 stores          // 상태 관리 (Zustand)  
├─ 📂 styles    
├─ 📂 types     
└─ 📂 utils  
```
